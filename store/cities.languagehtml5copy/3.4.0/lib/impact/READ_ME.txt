****************Steps to add unminified game code to project*****************

First, go to your downloads page on www.impactjs.com and download the newest version.
Second, unzip that bundle and copy all the files in lib/impactinto this folder. 
Third, open index.js in the root folder and change line 1 to AppMobi.context.include( 'lib/impact/impact.js' );

From here you can edit the engine code as wanted.
